## GET /api/v1/alerts

최근 7일 알림 목록 조회

### Request

| 파라미터 | 타입 | 필수 | 설명 |
|----------|------|------|------|
| category | String | N | 카테고리 필터 (all/ETF/PORTFOLIO/NEWS) |

### Response

```json
{
  "success": true,
  "data": {
    "alerts": [
      {
        "id": 123,
        "alertTypeCode": "ETF_LISTING",
        "alertTypeName": "ETF 신규 상장",
        "category": "ETF",
        "referenceType": "ETF",
        "referenceId": 45,
        "title": "KODEX 반도체 신규 상장",
        "message": "KODEX 반도체 ETF가 상장되었습니다.",
        "isRead": false,
        "createdAt": "2025-03-10T14:30:00"
      }
    ],
    "unreadCount": 12
  }
}
```

### 카테고리

| category | 설명 |
|----------|------|
| all | 전체 (기본값) |
| ETF | ETF 관련 알림 |
| PORTFOLIO | 포트폴리오 관련 알림 |
| NEWS | 뉴스 관련 알림 |

### 화면 이동 (referenceType 기준)

| referenceType | 이동 화면 | 파라미터 |
|---------------|----------|----------|
| ETF | ETF 상세 | `/etf/{referenceId}` |
| PORTFOLIO | 포트폴리오 상세 | `/portfolio/{referenceId}` |
| NEWS | 뉴스 상세 | `/news/{referenceId}` |
| null | 이동 없음 | - |

## GET /api/v1/alerts/unread/count

읽지 않은 알림 수 조회

### Response

```json
{
  "success": true,
  "data": {
    "count": 12
  }
}
```

### Request

| 파라미터 | 타입 | 필수 | 설명 |
|----------|------|------|------|
| alertId | Long | Y | 알림 ID (Path) |

### Response

```json
{
  "success": true
}
```

```json
        "groupName": "ETF 상장 알림",

        "isEnabled": false
      },
      {
        "settingGroup": "PORTFOLIO_RETURN",
        "groupName": "포트폴리오 수익률 알림",
        "description": "수익률에 큰 변화가 생기면 즉시 알려드려요.",

```

### 설정 그룹 (settingGroup)

| settingGroup | 화면 표시명 | 내부 alert_type |
|--------------|------------|-----------------|
| APP_NOTIFICATION | 앱 알림 | SYSTEM_ANNOUNCEMENT |
| ETF_LISTING | ETF 상장 알림 | ETF_LISTING |
| ETF_DELISTING | ETF 상장 폐지 알림 | ETF_DELISTING_SCHEDULED, ETF_DELISTING_COMPLETED |
| PORTFOLIO_REBALANCING | 포트폴리오 리밸런싱 알림 | PORTFOLIO_REBALANCING_SCHEDULED, PORTFOLIO_REBALANCING_COMPLETED |
| PORTFOLIO_RETURN | 포트폴리오 수익률 알림 | PORTFOLIO_RETURN_5PCT, PORTFOLIO_RETURN_10PCT, PORTFOLIO_LOSS_5PCT, PORTFOLIO_LOSS_10PCT |
| NEWS_NOTIFICATION | 뉴스 수신 알림 | NEWS_ETF_RELATED, NEWS_PORTFOLIO_RELATED |
