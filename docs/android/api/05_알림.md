# 알림 API

> 화면: `NotificationScreen.kt`, `NotificationSettingsScreen.kt`

---

## GET /api/v1/alerts

알림 목록 조회

### Request

| 파라미터 | 타입 | 필수 | 설명 |
|----------|------|------|------|
| page | int | N | 페이지 번호 (기본 0) |
| size | int | N | 페이지 크기 (기본 20) |
| category | String | N | 카테고리 필터 (all/ETF/PORTFOLIO/NEWS/SYSTEM) |

### Response

```json
{
  "success": true,
  "data": {
    "alerts": [
      {
        "id": 123,
        "alertTypeCode": "ETF_PRICE_CHANGE",
        "alertTypeName": "ETF 가격 변동",
        "category": "ETF",
        "referenceType": "ETF",
        "referenceId": 45,
        "title": "KODEX 반도체 급등",
        "message": "KODEX 반도체가 5% 이상 상승했습니다.",
        "isRead": false,
        "createdAt": "2025-03-10T14:30:00"
      }
    ],
    "page": 0,
    "totalPages": 5,
    "totalElements": 89,
    "unreadCount": 12
  }
}
```

### 카테고리

| category | 설명 |
|----------|------|
| all | 전체 (기본값) |
| ETF | ETF 관련 알림 |
| PORTFOLIO | 포트폴리오 관련 알림 |
| NEWS | 뉴스 관련 알림 |
| SYSTEM | 시스템 알림 |

```json
}
```

## POST /api/v1/alerts/fcm/token

FCM 토큰 등록 (푸시 알림용)

### Request

```json
{
  "token": "FCM_DEVICE_TOKEN_STRING",
  "deviceType": "ANDROID"
}
```

| 필드 | 타입 | 필수 | 설명 |
|------|------|------|------|
| token | String | Y | FCM 토큰 (최대 500자) |
| deviceType | String | Y | 기기 유형: ANDROID / IOS |

### Response

```json
{
  "success": true,
  "message": "FCM 토큰이 등록되었습니다."
}
```

## PUT /api/v1/alerts/settings

```json
  ]

```

```json
{
  "success": true,
  "data": {
    "updatedCount": 2
  }
}
```
