## GET /api/v1/alerts

최근 7일 알림 목록 조회

### Request

| 파라미터 | 타입 | 필수 | 설명 |
|----------|------|------|------|
| category | String | N | 카테고리 필터 (all/ETF/PORTFOLIO/NEWS) |

### Response

```json
{
  "success": true,
  "data": {
    "alerts": [
      {
        "id": 123,
        "alertTypeCode": "ETF_PRICE_CHANGE",
        "alertTypeName": "ETF 가격 변동",
        "category": "ETF",
        "referenceType": "ETF",
        "referenceId": 45,
        "title": "KODEX 반도체 급등",
        "message": "KODEX 반도체가 5% 이상 상승했습니다.",
        "isRead": false,
        "createdAt": "2025-03-10T14:30:00"
      }
    ],
    "unreadCount": 12
  }
}
```

### 카테고리

| category | 설명 |
|----------|------|
| all | 전체 (기본값) |
| ETF | ETF 관련 알림 |
| PORTFOLIO | 포트폴리오 관련 알림 |
| NEWS | 뉴스 관련 알림 |

### 화면 이동 (referenceType 기준)

| referenceType | 이동 화면 | 파라미터 |
|---------------|----------|----------|
| ETF | ETF 상세 | `/etf/{referenceId}` |
| PORTFOLIO | 포트폴리오 상세 | `/portfolio/{referenceId}` |
| NEWS | 뉴스 상세 | `/news/{referenceId}` |
| null | 이동 없음 | - |

## GET /api/v1/alerts/unread/count

읽지 않은 알림 수 조회

필요한 경우:
  - 홈 화면에서 알림 아이콘 뱃지만 표시할 때 (목록 안 불러오고 숫자만)
  - 주기적으로 폴링해서 새 알림 있는지 체크할 때

### Response

```json
{
  "success": true,
  "data": {
    "count": 12
  }
}
```

```json
}
```

## POST /api/v1/alerts/fcm/token

FCM 토큰 등록 (푸시 알림용)

### Request

```json
{
  "token": "FCM_DEVICE_TOKEN_STRING",
  "deviceType": "ANDROID"
}
```

| 필드 | 타입 | 필수 | 설명 |
|------|------|------|------|
| token | String | Y | FCM 토큰 (최대 500자) |
| deviceType | String | Y | 기기 유형: ANDROID / IOS |

### Response

```
HTTP 201 Created
```

```json
{
  "success": true,
  "message": "FCM 토큰이 등록되었습니다."
}
```

## PUT /api/v1/alerts/settings

알림 설정 수정

### Request

```json
{
  "settings": [
    {"alertTypeCode": "ETF_LISTING", "isEnabled": true},
    {"alertTypeCode": "NEWS_ALERT", "isEnabled": false}
  ]
}
```

### Response

```json
{
  "success": true,
  "data": {
    "settings": [
      {
        "alertTypeCode": "APP_NOTICE",
        "alertTypeName": "앱 알림",
        "category": "SYSTEM",
        "isEnabled": true
      },
      {
        "alertTypeCode": "ETF_LISTING",
        "alertTypeName": "ETF 상장 알림",
        "category": "ETF",
        "isEnabled": true
      },
      {
        "alertTypeCode": "NEWS_ALERT",
        "alertTypeName": "뉴스 수신 알림",
        "category": "NEWS",
        "isEnabled": false
      }
    ]
  }
}
```
