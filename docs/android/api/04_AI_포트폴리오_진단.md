## POST /api/v1/ai/portfolio/review

포트폴리오 AI 진단 요청

### Request

```json
{
  "portfolioSnapshotId": null,
  "portfolio": {
    "totalAmount": 10000000,
    "investmentType": "LUMP_SUM",
    "etfs": [
      {
        "ticker": "091160",
        "name": "KODEX 반도체",
        "weight": 40.0
      },
      {
        "ticker": "305720",
        "name": "KODEX 2차전지산업",
        "weight": 35.0
      },
      {
        "ticker": "364690",
        "name": "KODEX 미국S&P500TR",
        "weight": 25.0
      }
    ]
  }
}
```

### Request 필드

| 필드 | 타입 | 필수 | 설명 |
|------|------|------|------|
| portfolioSnapshotId | Long | N | 기존 스냅샷 ID (없으면 null) |
| portfolio.totalAmount | Long | Y | 총 투자금액 (1만원 ~ 100억원) |
| portfolio.investmentType | String | Y | 투자방식: `LUMP_SUM`(일시불) / `DCA`(정액) |
| portfolio.etfs | Array | Y | ETF 목록 |
| portfolio.etfs[].ticker | String | Y | ETF 티커 |
| portfolio.etfs[].name | String | Y | ETF 이름 |
| portfolio.etfs[].weight | Double | Y | 비중 (%) - 합계 100% |

### Response

```
HTTP 201 Created
```

```json
{
  "success": true,
  "message": "Success",
  "code": "OK",
  "data": {
    "reviewId": 12345,
    "headline": "공격적인 수익 추구!",
    "subHeadline": "기술주 중심의 로켓 포트폴리오",
    "keywords": ["기술주집중", "고변동성", "성장중심", "미국비중높음"],
    "analysis": "이 포트폴리오는 반도체와 2차전지 등 성장 섹터에 집중되어 있습니다. 높은 수익을 기대할 수 있지만, 시장 변동성에 취약할 수 있습니다. 미국 S&P500 ETF가 25%로 분산 효과를 주고 있으나, 전체적으로 공격적인 성향입니다. 변동성을 줄이려면 채권형 ETF나 배당 ETF 추가를 고려해보세요.",
    "llmModel": "gpt-4o",
    "createdAt": "2025-03-10T14:30:00"
  },
  "timestamp": "2025-03-10T14:30:00"
}
```

### 화면 매핑 (AiDiagnosisDialog)

| API 필드 | 화면 표시 |
|----------|----------|
| headline | 메인 타이틀 (큰 글씨) |
| subHeadline | 서브 타이틀 (작은 글씨) |
| keywords | 해시태그 칩 (#기술주집중 등) |
| analysis | 요약 상세 박스 내 텍스트 |

### 화면 상태

| 상태 | 표시 |
|------|------|
| Loading | CircularProgressIndicator + "AI가 포트폴리오를 분석 중입니다..." |
| Success | 진단 결과 카드 |
| Error | "진단 결과를 불러오지 못했습니다.\n다시 시도해주세요." |

## GET /api/v1/ai/portfolio/review/{reviewId}

리뷰 결과 조회 (기존 진단 결과 다시 보기)

### Request

| 파라미터 | 타입 | 필수 | 설명 |
|----------|------|------|------|
| reviewId | Long | Y | 리뷰 ID |

### Response

POST 응답과 동일

---

## GET /api/v1/ai/portfolio/reviews

리뷰 히스토리 조회

### Request

| 파라미터 | 타입 | 필수 | 설명 |
|----------|------|------|------|
| page | int | N | 페이지 번호 (기본 0) |
| size | int | N | 페이지 크기 (기본 10) |

### Response

```json
{
  "success": true,
  "message": "Success",
  "code": "OK",
  "data": {
    "reviews": [
      {
        "reviewId": 12345,
        "headline": "공격적인 수익 추구!",
        "subHeadline": "기술주 중심의 로켓 포트폴리오",
        "keywords": ["기술주집중", "고변동성", "성장중심"],
        "createdAt": "2025-03-10T14:30:00"
      }
    ],
    "page": 0,
    "totalPages": 5,
    "totalElements": 42
  },
  "timestamp": "2025-03-10T14:30:00"
}
```
