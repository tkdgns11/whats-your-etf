## POST /api/v1/auth/oauth/kakao

카카오 SDK에서 받은 access_token으로 로그인/자동 회원가입

### Request

| 파라미터 | 타입 | 필수 | 위치 | 설명 |
|----------|------|------|------|------|
| accessToken | String | Y | Body | 카카오 SDK Access Token |

```json
{
  "accessToken": "카카오_SDK_ACCESS_TOKEN"
}
```

### Response

```
HTTP 201 Created (신규 회원) / HTTP 200 OK (기존 회원)
```

```json
{
  "success": true,
  "message": "로그인 성공",
  "code": "OK",
  "data": {
    "accessToken": "eyJhbGciOiJIUzI1NiIs...",
    "refreshToken": "eyJhbGciOiJIUzI1NiIs...",
    "expiresIn": 3600,
    "isNewUser": true,
    "user": {
      "id": 1,
      "email": "user@kakao.com",
      "nickname": "홍길동",
      "profileImage": "https://k.kakaocdn.net/...",
      "loginProvider": "KAKAO"
    }
  },
  "timestamp": "2025-03-10T14:30:00"
}
```

### 필드 설명

| 필드 | 타입 | 설명 |
|------|------|------|
| accessToken | String | JWT Access Token |
| refreshToken | String | JWT Refresh Token |
| expiresIn | Int | Access Token 만료 시간 (초) |
| isNewUser | Boolean | 신규 회원 여부 (자동 가입 시 true) |
| user.id | Long | 사용자 ID |
| user.email | String | 이메일 |
| user.nickname | String | 닉네임 |
| user.profileImage | String? | 프로필 이미지 URL (nullable) |
| user.loginProvider | String | 로그인 방식 ("KAKAO") |

### 에러 코드

| 코드 | 메시지 | 설명 |
|------|--------|------|
| AUTH_001 | 카카오 인증 실패 | 유효하지 않은 access_token |

## POST /api/v1/auth/fcm/token

FCM 토큰 등록 (푸시 알림용, 로그인 직후 호출)

### Request

| 파라미터 | 타입 | 필수 | 위치 | 제약조건 | 설명 |
|----------|------|------|------|----------|------|
| token | String | Y | Body | 최대 500자 | FCM 토큰 |
| deviceType | String | Y | Body | ANDROID / IOS | 기기 유형 |

```json
{
  "token": "FCM_DEVICE_TOKEN_STRING",
  "deviceType": "ANDROID"
}
```

### Response

```
HTTP 201 Created
```

```json
{
  "success": true,
  "code": "OK",
  "message": "FCM 토큰이 등록되었습니다.",
  "timestamp": "2025-03-10T14:30:00"
}
```

### 참고

- **로그아웃 시 FCM 토큰 자동 삭제**: `POST /api/v1/auth/logout` 호출 시 백엔드에서 자동 처리
- 별도 DELETE 엔드포인트 불필요
