## 1. GET /api/v1/news

뉴스 목록 조회 (카테고리별 최신 20개)

### Request

| 파라미터 | 타입 | 필수 | 위치 | 기본값 | 설명 |
|----------|------|------|------|--------|------|
| category | String | N | Query | null | 뉴스 카테고리 코드. 미입력 시 전체 카테고리 조회 |

**카테고리 코드 (투자테마 13개 + 시장/경제):**

| 코드 | 이름 | 예시 |
|------|------|------|
| NEWS_SEMI | 반도체 | `/api/v1/news?category=NEWS_SEMI` |
| NEWS_IT | IT/전자 | `/api/v1/news?category=NEWS_IT` |
| NEWS_BIO | 바이오/의약 | |
| NEWS_AUTO | 자동차 | |
| NEWS_CHEM | 화학/소재 | |
| NEWS_ENERGY | 에너지 | |
| NEWS_FINANCE | 금융 | |
| NEWS_CONSTRUCT | 건설/부동산 | |
| NEWS_CONSUMER | 소비재 | |
| NEWS_TELECOM | 통신/미디어 | |
| NEWS_TRANSPORT | 운송/물류 | |
| NEWS_INDUSTRY | 산업재 | |
| NEWS_ETC | 기타 | |
| NEWS_MARKET | 시장/경제 | |

**요청 예시:**
```
GET /api/v1/news                       # 전체 뉴스 (최신 20개)
GET /api/v1/news?category=NEWS_SEMI    # 반도체 뉴스만 (최신 20개)
```

### Response

```json
{
  "success": true,
  "message": "Success",
  "code": "OK",
  "data": {
    "news": [
      {
        "id": 123,
        "title": "삼성전자, AI 반도체 대규모 투자 발표",
        "source": "한국경제",
        "thumbnailUrl": "https://...",
        "categoryCode": "NEWS_SEMI",
        "categoryName": "반도체",
        "publishedAt": "2025-03-10T09:30:00"
      }
    ]
  },
  "timestamp": "2025-03-10T14:30:00"
}
```

### 화면 매핑

| API 필드 | 화면 표시 |
|----------|----------|
| title | 뉴스 제목 (2줄 max) |
| source | 언론사명 |
| thumbnailUrl | 썸네일 이미지 |
| publishedAt | "3시간 전" 형식으로 변환 |
| categoryName | 카테고리 칩 |

## 3. GET /api/v1/news/search

뉴스 검색 (최신 20개)

### Request

| 파라미터 | 타입 | 필수 | 위치 | 제약조건 | 설명 |
|----------|------|------|------|----------|------|
| keyword | String | Y | Query | 2~50자 | 검색어 (제목+본문 LIKE 검색) |

**요청 예시:**
```
GET /api/v1/news/search?keyword=삼성전자
GET /api/v1/news/search?keyword=반도체
```

### Response

뉴스 목록과 동일한 구조 + `keyword` 필드 추가

```json
{
  "success": true,
  "message": "Success",
  "code": "OK",
  "data": {
    "news": [...],
    "keyword": "삼성전자"
  },
  "timestamp": "2025-03-10T14:30:00"
}
```

## 4. GET /api/v1/news/portfolio/{portfolioId}

포트폴리오 관련 뉴스 조회 (매일 오전 9시 캐시 갱신)

### Request

| 파라미터 | 타입 | 필수 | 위치 | 설명 |
|----------|------|------|------|------|
| portfolioId | Long | Y | Path | 포트폴리오 ID |

**요청 예시:**
```
GET /api/v1/news/portfolio/1
```

### Response

```json
{
  "success": true,
  "message": "Success",
  "code": "OK",
  "data": {
    "portfolioId": 123,
    "portfolioName": "기술주 중심 성장 전략",
    "news": [
      {
        "id": 456,
        "title": "미국 ETF 시장, 인공지능 테마로 자금 유입 역대 최고 기록",
        "summary": "글로벌 AI 경쟁 가속화로 관련 ETF 수익률 고공행진...",
        "source": "금융경제뉴스",
        "thumbnailUrl": "https://...",
        "publishedAt": "2025-03-10T09:30:00"
      }
    ],
    "updatedAt": "2025-03-10T09:00:00"
  },
  "timestamp": "2025-03-10T14:30:00"
}
```

### 로직

1. 포트폴리오 구성 ETF의 모든 구성종목 조회
2. 해당 종목들과 관련된 뉴스 매핑
3. **관련성 스코어**로 정렬 후 상위 5개 반환
4. 매일 오전 9시 캐시 갱신

### 선택 및 정렬 (2단계)

**1단계: 관련 뉴스 5개 선택 (스코어 기준)**
```
score = relevance × recency_multiplier

relevance = 포트폴리오_ETF_비중 × ETF_종목_비중
recency_multiplier = 1 / (1 + 경과일수 × 0.3)
```

| 경과일 | recency_multiplier | 예시 (relevance=300) |
|--------|-------------------|---------------------|
| 0일 | 1.0 | 300 |
| 1일 | 0.77 | 231 |
| 3일 | 0.53 | 159 |
| 7일 | 0.32 | 96 |

**2단계: 최종 정렬 (최신순)**
- 선택된 5개 뉴스를 `published_at DESC`로 정렬하여 반환

```
news_stock_mapping (뉴스 → 종목)
    → stock.company_id (종목 → 회사)
    → etf_stock_composition (회사 → ETF 구성)
    → etf (ETF 정보)
    → etf_prices (최신 시세/등락률)
```

- 비중(weight_pct) 높은 ETF 순으로 최대 5개 반환
- 등락률은 최신 거래일 기준
