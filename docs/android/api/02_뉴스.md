## 1. GET /api/v1/news

뉴스 목록 조회 (카테고리 필터, 페이징)

### Request

| 파라미터 | 타입 | 필수 | 설명 |
|----------|------|------|------|
| page | Int | N | 페이지 번호 (기본: 0) |
| size | Int | N | 페이지 크기 (기본: 20, 최대: 100) |
| category | String | N | 카테고리 코드 (없으면 전체) |

**카테고리 코드:**
- `NEWS_MACRO` - 금리/거시경제
- `NEWS_SEMI` - 반도체
- `NEWS_IT` - AI/테크
- `NEWS_DIVIDEND` - 배당
- `NEWS_BATTERY` - 2차전지
- `NEWS_GLOBAL` - 글로벌

### Response

```json
{
  "success": true,
  "data": {
    "news": [
      {
        "id": 123,
        "title": "삼성전자, AI 반도체 대규모 투자 발표",
        "summary": "삼성전자가 AI 반도체에 10조원 투자...",
        "source": "한국경제",
        "thumbnailUrl": "https://...",
        "publishedAt": "2025-03-10T09:30:00",
        "categoryCode": "NEWS_SEMI",
        "categoryName": "반도체"
      }
    ],
    "page": 0,
    "totalPages": 10,
    "totalElements": 195
  }
}
```

### 화면 매핑

| API 필드 | 화면 표시 |
|----------|----------|
| title | 뉴스 제목 (2줄 max) |
| summary | 요약 (1줄) |
| source | 언론사명 |
| thumbnailUrl | 썸네일 이미지 |
| publishedAt | "3시간 전" 형식으로 변환 |
| categoryName | 카테고리 칩 |

## 2. GET /api/v1/news/{newsId}

뉴스 상세 조회 (AI 요약 + 관련 ETF 포함)

### Response

```json
{
  "success": true,
  "data": {
    "id": 123,
    "title": "삼성전자, AI 반도체 대규모 투자 발표",

    "source": "한국경제",
    "sourceUrl": "https://www.hankyung.com/...",
    "thumbnailUrl": "https://...",
    "publishedAt": "2025-03-10T09:30:00",
    "categoryCode": "NEWS_SEMI",
    "categoryName": "반도체",
    "keywords": ["AI반도체", "삼성전자", "투자확대", "HBM"],
    "aiSummary": [

      "경쟁사 대비 기술 격차 확대 전망"
    ],
    "relatedStocks": [
      {
        "companyId": 1,
        "ticker": "005930",
        "name": "삼성전자"
      }
    ],
    "relatedEtfs": [
      {
        "etfId": 45,
        "ticker": "091160",
        "name": "KODEX 반도체",
        "manager": "삼성자산운용",
        "changeRate": 1.24
      },
      {
        "etfId": 46,
        "ticker": "091180",
        "name": "TIGER 반도체",
        "manager": "미래에셋자산운용",
        "changeRate": 1.56
      }
    ]
  }
}
```

### 화면 매핑

| API 필드 | 화면 표시 |
|----------|----------|
| title | 헤더 제목 |
| thumbnailUrl | 헤더 배경 이미지 |
| publishedAt | 날짜 표시 |
| keywords | 해시태그 칩 (#금리동결 등) |
| aiSummary | AI 핵심 요약 박스 (bullet points) |
| content | 본문 텍스트 |
| sourceUrl | "원문 전체 보러가기" 링크 |
| relatedEtfs | 관련 ETF 카드 (가로 스크롤) |

### 관련 ETF 카드 필드

| 필드 | 화면 표시 |
|------|----------|
| ticker | ETF 티커 (원형 배지) |
| name | ETF 이름 |
| manager | 운용사 |
| changeRate | 등락률 (초록/빨강) |

## 3. GET /api/v1/news/search

뉴스 검색

### Request

| 파라미터 | 타입 | 필수 | 설명 |
|----------|------|------|------|
| keyword | String | Y | 검색어 (2~50자) |
| page | Int | N | 페이지 번호 |
| size | Int | N | 페이지 크기 |

### Response

뉴스 목록과 동일 + `keyword` 필드 추가

### relatedEtfs 조회 로직

```
news_stock_mapping (뉴스 → 종목)
    → stock.company_id (종목 → 회사)
    → etf_stock_composition (회사 → ETF 구성)
    → etf (ETF 정보)
    → etf_prices (최신 시세/등락률)
```

- 비중(weight_pct) 높은 ETF 순으로 최대 5개 반환
- 등락률은 최신 거래일 기준
